## ✅ 正确的稀有度特征

根据真实截图，总结出各稀有度的准确特征：

| 稀有度  | Cost  | 外框颜色         | 强度             | 效果特点                                             | 示例数量             |
| ------- | ----- | ---------------- | ---------------- | ---------------------------------------------------- | -------------------- |
| **R**   | -5~-2 | **白色**         | 弱               | 极简单、单一效果                                     | 1个效果              |
| **SR**  | -5~-2 | **黄色**         | 中等             | 2-3个效果的组合 或 带条件的强力单一效果              | 2-4个效果            |
| **SSR** | -5~-1 | **彩色（渐变）** | 强 或 战略性     | 多个强力效果的组合 或 条件性超强效果 或 消耗Buff触发 | 3-5个效果            |
| **UR**  | -6~-1 | **彩虹/特殊**    | 压倒性 或 颠覆性 | 改变规则的机制 或 极其强大的复合效果                 | 5+个效果 或 独特机制 |

## 🔍 关键发现

### 1. **Cost区间正确认知**

- ❌ **错误**：之前以为是正数Cost（1-6费）
- ✅ **正确**：实际是**负数Cost**（-1~-6费）
  - R卡：-5~-2费
  - SR卡：-5~-2费
  - SSR卡：-5~-1费
  - UR卡：-6~-1费（推测）

### 2. **外框颜色识别**

- ✅ **R卡**：白色外框
- ✅ **SR卡**：黄色外框
- ✅ **SSR卡**：彩色渐变外框（粉→蓝→黄等）
- ✅ **UR卡**：需自行设计（建议彩虹或特殊发光效果）

### 3. **效果复杂度**

从真实截图可见：
- **R卡最简单**：仅1个效果（如"提升好印象150%的数值"）
- **SR卡中等**：2-4个效果组合（如"干劲消耗3 + 提升好印象90% + 技能卡使用数+1"）
- **SSR卡复杂**：3-5个效果组合，或消耗Buff触发超强效果
- **UR卡极端**：需要自行创造，建议5+个效果或改变游戏规则

### 4. **非凡系统的特殊性**

非凡系统拥有独特的关键词和机制：
- **全力状态**、**温存状态**、**坚决状态**
- **热意值**（类似感性的好印象、理性的集中）
- **状态切换次数计数**（如"切换温存状态两次及以上"）
- **全局效果**（如"所有技能卡的消耗值-1"）

## ✨ UR卡创造思路

根据SSR卡的效果，UR卡应该具有以下特征之一或多个：

### 思路1：SSR效果的2倍强度
```
SSR示例：国民的偶像（-1费）
- 消耗状态良好1回合
- 下一张卡效果发动2次
- 技能卡使用数+1

UR升级版：传说的偶像（-1费）
- 消耗状态良好2回合
- 下两张卡效果各发动3次
- 技能卡使用数+2
- 所有Buff持续时间+1回合
```

### 思路2：改变游戏规则
```
UR示例：时光倒流（-6费）
- 消耗全部体力
- 回合数+2
- 获得状态绝佳+5回合
- 复制手牌中所有卡
- 课程中限1次，不可重复
```

### 思路3：极其复杂的联动
```
UR示例：完美演出（-3费）
- 消耗状态良好、好调、状态绝佳各1回合
- 全属性Buff同时触发
- 所有技能卡Cost-2（本回合）
- 技能卡使用数+3
- 下回合手牌+3
- 课程中限1次，不可重复
```
