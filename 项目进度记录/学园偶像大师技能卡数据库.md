**注意**：文件名已移除所有非法字符 `\/:*?"<>|`，使用时需要进行相应处理。

### 构建URL示例（TypeScript）
```typescript
function buildSkillCardImageUrl(cardName: string): string {
  // 移除Windows非法字符
  const sanitizedName = cardName.replace(/[\\/:"*?<>|]/g, '');
  const baseUrl = 'https://raw.githubusercontent.com/2426269/shinycolors-assets-cdn/main/技能卡卡面';
  return `${baseUrl}/${encodeURIComponent(sanitizedName)}.webp`;
}
```

## 🎯 AI生成规则总结

### 1. **元气消耗 (Cost)**

从实际数据可见，元气消耗范围：
- **N卡**: 0-7
- **R卡**: 0-7
- **SR卡**: 0-7
- **SSR卡**: 0-12

**特殊情况**：
- 有些卡消耗其他资源（体力、好印象、干劲、专注等）而不消耗元气
- 强化后可能改变Cost（如从7费→-6费）

### 2. **效果数量**

从实际数据统计：
- **N卡**: 1-2个效果
- **R卡**: 1-2个效果
- **SR卡**: 2-4个效果
- **SSR卡**: 3-6个效果

### 3. **属性关键词偏向**

#### 非凡（Anomaly）
- **核心关键词**：全力值、温存、強気、熱意追加
- **特殊机制**：指針固定、保留移动、全力状态条件触发
- **典型效果**：状态切换、全力值操作、卡牌位置操作

#### 理性（Logic）
- **核心关键词**：好印象、やる気（干劲）、元気
- **特殊机制**：百分比转换、消耗属性触发
- **典型效果**：属性叠加、百分比参数上昇

#### 感性（Sense）
- **核心关键词**：状態良好、好調、集中、専注、活力
- **特殊机制**：消耗Buff触发、长期Buff叠加
- **典型效果**：Buff管理、状态良好消耗、卡牌效果翻倍

#### 自由（Free）
- **核心关键词**：混合多种属性
- **特殊机制**：不受培育计划限制
- **典型效果**：多属性平衡提升

### 4. **特殊效果类型**

从实际数据中提取的特殊效果：
1. **卡牌操作**：
   - 手札を選択し、保留に移動
   - 山札か捨札にあるスキルカードを選択し、保留に移動
   - スキルカードを引く

2. **使用数增加**：
   - スキルカード使用数追加+1
   - 次に使用したスキルカードの効果を追加発動

3. **改变规则**：
   - ターン追加+1（回合数+1）
   - 指針固定+Nターン
   - レッスン開始時手札に入る

4. **百分比转换**：
   - 元気のX%分パラメータ上昇
   - やる気のX%分パラメータ上昇
   - 好印象のX%分パラメータ上昇

5. **消耗触发**：
   - 好印象消費N
   - やる気消費N
   - 状態良好消費Nターン
   - 専注消費N

6. **体力操作**：
   - 体力消費N
   - 体力回復N
   - 消費体力減少Nターン

### 5. **课程限制**

绝大多数SSR/SR卡都带有以下限制：
- `※レッスン中1回`（课程中限1次）
- `※重複不可`（不可重复）

1. **先选择属性**（非凡/理性/感性/自由）
2. **再选择稀有度**（N/R/SR/SSR）
3. **参考该属性+稀有度的真实示例**
4. **遵循效果数量规则**
5. **使用该属性的关键词**
