## 🎯 修正目标

根据用户反馈，纠正以下关键概念的翻译和理解：

1. **"重複不可"和"使用可能"** - 之前理解错误
2. **"レッスン中1回"** - 翻译和机制理解有误

## ❌ 修正前的错误理解

### 错误 1："重複不可"理解错误

**之前的理解**：
```
"重複不可" = 训练中只能使用1次，用完后从牌库移除
```

**实际含义**：
```
"重複不可" = 卡组里只能有一张这个卡（不可重复获得）
```

---

### 错误 2："レッスン中1回"翻译错误

**之前的翻译**：
```
"レッスン中1回" = "训练中1回"
```

**正确的翻译**：
```
"レッスン中1回" = "演出中限1次"（レッスン = Lesson = 课程/演出）
```

---

### 错误 3：对卡牌使用机制的理解不完整

**之前没有明确区分**：
- 手牌（手札）
- 牌组（山札）
- 弃牌堆（捨札）
- 除外堆

---

## ✅ 修正后的正确理解

### 1. 卡牌获取限制

| 术语     | 日文     | 含义                               |
| -------- | -------- | ---------------------------------- |
| 重複不可 | 重複不可 | 卡组里只能有一张这个卡             |
| 使用可能 | 使用可能 | 卡组可以有多个这个卡（可重复获得） |

**说明**：
- "重複不可"限制的是**卡组构筑**，不是使用次数
- "使用可能"表示可以在卡组中放入多张

---

### 2. 卡牌使用限制

| 限制类型    | 日文          | 机制说明                                                       |
| ----------- | ------------- | -------------------------------------------------------------- |
| 演出中限1次 | レッスン中1回 | 一场演出中只能使用一次，使用后进入"除外"堆，下一场演出依旧可用 |
| 无限制      | （无标注）    | 可以多次使用，使用后进入"弃牌堆"，可以再次抽取使用             |

**卡牌流程示例**：

#### 有"演出中限1次"的卡：
```
牌组（山札）
  ↓ 抽取
手牌（手札）
  ↓ 使用
除外堆
  ↓ 下一场演出
牌组（山札）← 重新可用
```

#### 无限制的卡：
```
牌组（山札）
  ↓ 抽取
手牌（手札）
  ↓ 使用
弃牌堆（捨札）
  ↓ 牌组用尽时
牌组（山札）← 重新洗入
  ↓ 可以再次抽取
手牌（手札）← 同一场演出中可多次使用
```
