## 🔧 解决的问题

### 1. 文件名冲突解决
**问题**: 【雪空 セパレート】大崎甘奈 文件名不一致
- 本地版本: `雪空 セパレート 大崎甘奈.webp` ✅（正确）
- 远程版本: `雪空 セパレ ート 大崎甘奈.webp` ❌（空格位置错误）

**解决方案**:
```bash
git rm "角色卡面/雪空 セパレ ート 大崎甘奈.webp"  # 删除远程错误版本
git rm "角色卡面/雪空セパレ ート 大崎甘奈.webp"   # 删除原始错误版本
git add "角色卡面/雪空 セパレート 大崎甘奈.webp"   # 保留本地正确版本
git commit -m "解决冲突：统一雪空 セパレート 大崎甘奈文件名（正确空格位置）"
```

**结果**: 
- ✅ 文件名统一为 `雪空 セパレート 大崎甘奈.webp`
- ✅ 与觉醒图 `雪空 セパレート 大崎甘奈+.webp` 完美匹配

**解决方案**:

## 💡 经验总结

### 文件命名最佳实践
1. **空格规范**: 主题和角色之间用单个空格
2. **觉醒标记**: 用 `+` 后缀，紧接文件名
3. **扩展名**: 统一使用 `.webp`
4. **特殊符号**: 直接保留日文原字符（`...` 而非 `…`）

### Git操作经验
1. **推送前拉取**: 避免冲突
2. **冲突优先本地**: 确保文件名正确性
3. **提交信息详细**: 便于回溯
4. **分支跟踪**: 及时设置上游

### 数据管理策略
1. **多层备份**: JSON文件多版本备份
