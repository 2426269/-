## 🔍 故障排除

### 问题1: 推送被拒绝
```
! [rejected]        main -> main (fetch first)
```

**解决**:
```powershell
git pull --rebase
git push
```

### 问题2: 文件太大
```
remote: error: File size exceeds GitHub's file size limit of 100 MB
```

**解决**:
- 已使用WebP压缩，所有文件小于2MB
- 如果仍有问题，手动压缩大图片

### 问题3: CDN更新延迟
GitHub Raw可能需要5-10分钟更新缓存

**解决**:
- 等待5-10分钟后重试
- 或使用 `?t=timestamp` 强制刷新
